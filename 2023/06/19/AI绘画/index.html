Stable Diffusion的简介

目前AI绘画最火的当属 Midjorney 和 Stable Diffusion，Midjourney没有开源，Stable Diffusion 免费开源。Stable Diffusion 是StabilityAI公司于2022年提出的，它是一个基于潜在 扩散模型（Latent Diffusion Models），能够根据文本描述生成高质量、高分辨率、高逼真的图像。

至于其原理，不做过多介绍，如果想深入了解，可查看：

源码：https://github.com/CompVis/stable-diffusion

论文：https://arxiv.org/pdf/2112.10752.pdf

Stable Diffusion本身没有类似exe的可直接运行的软件，好在有一些基于 stable-diffusion 封装的 webui 开源项目，其中，AUTOMATIC1111大神 部署的框架 stable-diffusion-webui ，得到了广泛的应用。它是一套无代码、可视化的Stable Diffusion集成运行环境。让我们能通过web可视化界面而不是通过命令行参数使用Stable Diffusion进行绘画创作。

因为每个人的电脑环境不一样，本地部署免不了会出现各种各样的问题，如果不想折腾，可直接下载使用别人的整合包，下载了这些整合包，直接解压就可以使用，现在国内热度比较高的有两个整合包：

秋叶aaaki v4整合包：https://pan.baidu.com/s/13_B9lotU7UDcET7wYcDF7g?pwd=bang#list/path=%2F

独立研究员-星空整合包：https://pan.baidu.com/s/1_FYbDpkX6Q6hDVHJpCMRxA?pwd=o852

在了解之前，我是直接进行本地部署的，下面将详细介绍部署stable-diffusion-webui的具体流程。

系统配置需求

Stable Diffusion是比较耗资源的，因此对电脑硬件有一定要求。

NVIDIA GPU 至少 4GB 显存，至少8GB 内存，至少 20GB 可用硬盘空间，如果想要生成速度快，显卡配置自然是越高越好，显存最好也上到8G。如果本地机器配置达不到，可使用云端部署，可参考：https://b23.tv/KLk8gPg

环境准备

  • Python

请注意，为保证准确性，Python版本至少为 3.10.6，点击这里直接下载

  • git

为避免不必要的麻烦，git安装时一定要勾选 将PATH添加到系统环境变量中

  • stable-diffusion-webui库

打开 git bash ,输入命令 git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git ,下载stable-diffusion-webui库。

  • CUDA

stable-diffusion-webui 运行使用的是 GPU 算力，这里我们需要安装 CUDA 驱动，打开cmd命令行，输入 nvidia-smi, 查看电脑CUDA的版本号：

然后进入Nvidia CUDA 官网：https://developer.nvidia.com/cuda-toolkit-archive， 找到对应的系统版本号，下载安装。

  • 模型Stable Diffusion checkpoint

stable-diffusion-webui需要有已经训练过的模型才能进行使用，两个常用的下载模型网站：

huggingface：https://huggingface.co/

civitai ：https://civitai.com/

这里下载官方模型sd-v1-4

  • 网络控制扩展包ControlNet extension

打开 git bash ,输入命令 git clone https://github.com/Mikubill/sd-webui-controlnet.git ,下载sd-webui-controlnet extension.

  1. 依次安装以上python和git软件，CUDA软件。

  2. 将之前下载的模型（sd-v1-4.ckpt）放到 stable-diffusion-webui-diffusion下。将ControlNet extension文件夹放到 stable-diffusion-webui。

  3. 打开刚刚下载的 stable-diffusion-webui文件夹，找到如下bash文件，针对不同的电脑，双击不同的bat文件。

  1. 弹出命令窗口后，它会自动下载相关内容，注意：这一步可能会出现很多问题，根据提示，进行一一解决
