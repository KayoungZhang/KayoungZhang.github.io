Qt常用控件之组合框QComboBox

本文最后更新于：2023年7月3日 下午

Qt中组合框（QComboBox）提供了一种占用空间最小的方式，其表现形式如下：

属性

  • count: const int -->组合框中的项目数
  • currentData : const QVariant -->当前项目的数据
  • currentIndex : int -->组合框中当前项目的索引
  • currentText : QString -->当前文本
  • insertPolicy : InsertPolicy -->插入的项目在组合框中的位置策略
  • maxCount : int -->组合框中允许的最大项目数
  • minimumContentsLength : int -->组合框的最小字符数
  • modelColumn : int -->模型中可见的列
  • sizeAdjustPolicy : SizeAdjustPolicy -->组合框大小的更改策略

信号

void activated(int index)  //当选择组合框中的项目时发送此信号。
void currentIndexChanged(int index)  //当组合框中的 currentIndex更改时，就会发送此信号。如果组合框变空则index为 -1。 
void currentTextChanged(const QString &text)  //当组合框中的项目更改时，就会发送此信号。
void editTextChanged(const QString &text) //当组合框的行编辑小部件中的文本发生更改时，会发出此信号。

槽函数与成员函数

void clear()  //清除组合框，删除所有项目。
void clearEditText()  //清除组合框中用于编辑的行编辑的内容。
void setCurrentIndex(int index)  //设置当前项目索引
void setCurrentText(const QString &text) //设置当前项目文字
void setEditText(const QString &text)  //设置项目编辑的文字

/* 添加项目和删除项目及获取或设置项目 */
void addItem(const QString &text, const QVariant &userData = QVariant())
void addItem(const QIcon &icon, const QString &text, const QVariant &userData = QVariant())
void addItems(const QStringList &texts)    
void insertItem(int index, const QString &text, const QVariant &userData = QVariant())
void insertItem(int index, const QIcon &icon, const QString &text, const QVariant &userData = QVariant())
void insertItems(int index, const QStringList &list)
void removeItem(int index)
void setInsertPolicy(QComboBox::InsertPolicy policy) //设置插入策略
int currentIndex() const
QString currentText() const    
void setItemText(int index, const QString &text)

这里主要说一下增加项目的方法，既可以通过以上函数增加，也可以在UI设计界面进行增加

  • 通过函数
QStringList cityList = {"广州市", "深圳市", "珠海市", "东莞市"};
// 添加城市
ui->comboBox_province->addItem("广东省");
ui->comboBox_city->addItems(cityList);
  • 在组合框上右键->【编辑项目…】，在打开的编辑窗口中可以增加删除条目

案例

创建一个基类为QWidget的工程，如下：

下面进行工程实现：

Widget::Widget(QWidget *parent)
    : QWidget(parent)
    , ui(new Ui::Widget)
{
    ui->setupUi(this);

    /* 创建城市列表 */
    gdCityList = QStringList{"广州市", "深圳市", "珠海市"};
    hnCityList = QStringList{"郑州市", "洛阳市", "信阳市"};
    hbCityList = QStringList{"武汉市", "黄石市", "宜昌市"};
    sdCityList = QStringList{"济南市", "青岛市", "烟台市"};
    zjCityList = QStringList{"杭州市", "宁波市", "温州市"};
    /* 创建大学列表 */
    gdUniversityList = QStringList{"中山大学", "广州大学", "深圳大学", "南方科技大学"};
    hnUniversityList = QStringList{"郑州大学", "河南大学", "信阳师范大学", "河南科技大学"};
    hbUniversityList = QStringList{"武汉大学", "华中科技大学", "湖北师范大学", "三峡大学"};
    sdUniversityList = QStringList{"济南大学", "青岛大学", "青岛理工大学", "烟台大学"};
    zjUniversityList = QStringList{"浙江大学", "杭州师范大学", "宁波大学", "温州大学"};

    /* 为comboBox_province添加省份 */
    ui->comboBox_province->addItem(QIcon(":/pic/guangdong.png"), "广东省");
    ui->comboBox_province->addItem(QIcon(":/pic/henan.png"), "河南省");
    ui->comboBox_province->addItem(QIcon(":/pic/hubei.png"), "湖北省");
    ui->comboBox_province->addItem(QIcon(":/pic/shandong.png"), "山东省");
    ui->comboBox_province->addItem(QIcon(":/pic/zhejiang.png"), "浙江省");

    /* 为comboBox_city和comboBox_school添加默认城市和学校 */
    ui->comboBox_province->setCurrentIndex(0);
    ui->comboBox_city->addItems(gdCityList);
    ui->comboBox_school->addItems(gdUniversityList);
    /* 自定义槽函数，当省份改变，触发槽机制，连接信号与槽 */
    connect(ui->comboBox_province, SIGNAL(currentIndexChanged(int)), this, SLOT(province_changed_slot(int)));
}

/* 当省份改变时，城市和学校相应发生变化 */
void Widget::province_changed_slot(int index)
{
    ui->comboBox_city->clear();//清空之前的项目内容
    ui->comboBox_school->clear();
    switch (index)
    {
        case 0:
            ui->comboBox_city->addItems(gdCityList);
            ui->comboBox_school->addItems(gdUniversityList);
            break;
        case 1:
            ui->comboBox_city->addItems(hnCityList);
            ui->comboBox_school->addItems(hnUniversityList);
            break;
        case 2:
            ui->comboBox_city->addItems(hbCityList);
            ui->comboBox_school->addItems(hbUniversityList);
            break;
        case 3:
            ui->comboBox_city->addItems(sdCityList);
            ui->comboBox_school->addItems(sdUniversityList);
            break;
        case 4:
            ui->comboBox_city->addItems(zjCityList);
            ui->comboBox_school->addItems(zjUniversityList);
            break;
        default:
            break;
    }
}

void Widget::on_btn_clicked()
{
    /* 获取姓名、大学、省份、城市 */
    QString name = ui->comboBox_name->currentText();
    QString province = ui->comboBox_province->currentText();
    QString city = ui->comboBox_city->currentText();
    QString university = ui->comboBox_school->currentText();
    QString info = name + " 来自于" + province + city + "的" + university;

    QMessageBox::information(this, "学生信息", info);
}
