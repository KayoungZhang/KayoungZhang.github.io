Qt常用控件之微调框QSpinBox

本文最后更新于：2023年7月3日 下午

QSpinBox 用于整数的显示和输入，QDoubleSpinBox则用于显示和输入浮点值。它们均派生自QAbstractSPinBox类，具有大多数相同的属性，只是参数类型不同（一个 int，一个double）。

修改微调框数值的方式包括：

  • 单击右侧的向上/向下按钮
  • 按键盘的向上/向下键
  • 在微调框获取焦点时，通过鼠标滚轮的上下滚动

属性和方法

  • 获取和设置当前值
int value() const
void setValue(int val)
  • 获取和设置最大、最小值
int maximum() const
void setMaximum(int max)
int minimum() const
void setMinimum(int min)
    
void setRange(int minimum, int maximum)
  • 获取和设置步长
int singleStep() const
void setSingleStep(int val)
  • 获取和设置变化值是否循环改变
bool wrapping() const
void setWrapping(bool w)
  • 获取和设置是否加速数值变化速度
bool isAccelerated() const
void setAccelerated(bool on)
  • 获取和设置前缀和后缀
QString prefix() const
void setPrefix(const QString &prefix)
QString suffix() const
void setSuffix(const QString &suffix)
  • 信号
void textChanged(const QString &text)
void valueChanged(int i)

void valueChanged(double d)

实例

创建工程，进入UI设计界面，设计如下：

功能实现：

Widget::Widget(QWidget *parent)
    : QWidget(parent)
    , ui(new Ui::Widget)
{
    ui->setupUi(this);

    /* 设置SpinBox初始属性 */
    ui->spinBox_weight->setSuffix("kg"); //后缀
    ui->spinBox_weight->setRange(0, 100); //范围0~100
    ui->spinBox_weight->setSingleStep(1); //步长为1
    ui->spinBox_weight->setAccelerated(true); //允许加速
    ui->spinBox_weight->setWrapping(true); //允许循环

    connect(ui->doubleSpinBox_price, SIGNAL(valueChanged(double)), this, SLOT(price_value_slot(double)));
    connect(ui->spinBox_weight, SIGNAL(valueChanged(int)), this, SLOT(weight_value_slot(int)));
}

Widget::~Widget()
{
    delete ui;
}

void Widget::price_value_slot(double price)
{
    double total = ui->spinBox_weight->value() * price;
    ui->lineEdit_total->setText(QString::number(total) + "元");
}

void Widget::weight_value_slot(int weight)
{
    double total = ui->doubleSpinBox_price->value() * weight;
    ui->lineEdit_total->setText(QString::number(total) + "元");
}
