Qt常用控件之标签QLabel

本文最后更新于：2023年6月29日 晚上

QLabel属性和方法

QLabel用于显示文本或图像，它有很多属性和方法，常用的主要有：

  • 获取和设置文本
String text() const;
void setText(const QString &text);
  • 设置标签中内容的对齐方式
Qt::Alignment alignment() const;
void setAlignment(Qt::Alignment);

Qt::Alignment 是一个宏，常用取值如下：

  • 标签内容换行
bool wordWrap() const;
void setWordWrap(bool on);
  • 获取和设置标签上显示的图像
const QPixmap *pixmap() const;
void setPixmap(const QPixmap &);
  • 设置动图
void setMovie(QMovie *movie);

QLabel的使用

创建基类为QWidget的工程，进入UI设计界面，进行如下设计：

下面进行功能实现：

/* 缩放widget，标签文本允许换行 */
void Widget::on_btn_enableWrap_clicked()
{
    ui->label_text->setWordWrap(true);
}

/* 缩放widget，标签文本不允许换行 */
void Widget::on_btn_disableWrap_clicked()
{
    ui->label_text->setWordWrap(false);
}

/* 设置标签文本对齐方式 */
void Widget::on_btn_left_clicked()
{
    ui->label_text->setAlignment(Qt::AlignLeft | Qt::AlignVCenter);
}


void Widget::on_btn_center_clicked()
{
    ui->label_text->setAlignment(Qt::AlignCenter);
}


void Widget::on_btn_right_clicked()
{
    ui->label_text->setAlignment(Qt::AlignRight | Qt::AlignVCenter);
}
/* 弹出消息对话框信息 */
void Widget::on_btn_getText_clicked()
{
    QMessageBox::information(this, "文本信息", ui->label_text->text());//需要添加头文件<QMessageBox>
}


void Widget::on_btn_setText_clicked()
{
    /* 获取当前时间并转换为字符串 */
    QString str = QTime::currentTime().toString(); //需要添加头文件<QTime>
    ui->label_text->setText(str);
    QFont font = QFont("宋体", 15);
    ui->label_text->setFont(font); //设置字体
    ui->label_text->setAlignment(Qt::AlignCenter);
}


void Widget::on_btn_setlink_clicked()
{
    ui->label_text->setText("<h1><a href=\"https://www.baidu.com\">百度一下</a></h1>");
    /* 开启点击自动跳转到超链接地址 */
    ui->label_text->setOpenExternalLinks(true);
    ui->label_text->setAlignment(Qt::AlignCenter);
}


void Widget::on_btn_change_clicked()
{
    QPixmap pixmap;

    /* 首先判断图片切换的位置 */
    if(index > 12)

    }

void Widget::on_btn_disp_clicked()
{
    QMovie *movie = new QMovie(":/png/movie.gif");//添加头文件<QMovie>
    movie->start(); //播放动画
    ui->label_pic->setMovie(movie);
}
