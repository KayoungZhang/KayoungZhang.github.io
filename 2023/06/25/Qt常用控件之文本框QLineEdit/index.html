Qt常用控件之文本框QLineEdit

本文最后更新于：2023年6月29日 晚上

QLineEdit是一个单行文本输入框,允许用户输入和编辑单行纯文本，它自带一些如撤销、重做、剪切、粘贴、拖拽等文本编辑功能。

类型成员

enum ActionPosition { LeadingPoisition, TrailingPosition }

此枚举类型描述QLineEdit应该如何显示要添加的Action小部件。

  • QLineEdit::LeadingPosition：当使用 Qt::LeftToRight 布局方向时，小部件显示在文本的左侧，当使用 Qt::RightToLeft 时，小部件则显示在文本的右侧。
  • QLineEdit::TrailingPosition：当使用 Qt::LeftToRight 布局方向时，小部件显示在文本的右侧，当使用 Qt::RightToLeft 时，小部件则显示在文本的左侧。

enum EchoMode { Normal, NoEcho, Password, PasswordEchoOnEdit }

此枚举类型描述QLineEdit应如何显示其内容。

  • QLineEdit::Normal :输入时显示字符, 默认设置。
  • QLineEdit::NoEcho :不显示任何输入内容。
  • QLineEdit::Password ：输入字符一般以●号表示。
  • QLineEdit::PasswordEchoOnEdit ：在输入时显示输入的字符，输入完成后一般以●号显示。

void setAlignment(Qt::Alignment flag)
  • 获取和设置回显模式
QLineEdit::EchoMode echoMode() const
void setEchoMode(QLineEdit::EchoMode)

bool isReadOnly() const
void setReadOnly(bool)
  • 设置文本框是否使能
bool isEnabled() const
void setEnabled(bool)
  • 设置和获取文本框内容格式

新建一个 lineEdit 的工程，进入UI设计界面，设计如下：

下面进行功能实现：

void Widget::on_btn_print_clicked()
{
    QMessageBox::information(this, "信息输出", ui->lineEdit_content->text());
}

void Widget::on_btn_normal_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setEchoMode(QLineEdit::Normal);
    ui->lineEdit_content->setPlaceholderText("Normal");
}

void Widget::on_btn_noecho_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setEchoMode(QLineEdit::NoEcho);
    ui->lineEdit_content->setPlaceholderText("NoEcho");
}

void Widget::on_btn_password_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setEchoMode(QLineEdit::Password);
    ui->lineEdit_content->setPlaceholderText("Password");
}

void Widget::on_btn_pwdAndEcho_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setEchoMode(QLineEdit::PasswordEchoOnEdit);
    ui->lineEdit_content->setPlaceholderText("PasswordEchoOnEdit");
}

void Widget::on_btn_placeholder_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setPlaceholderText("请输入名字：");
}

void Widget::on_btn_clearbtn_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setClearButtonEnabled(true);
}

void Widget::on_btn_rw_clicked()
{
    ui->lineEdit_content->setReadOnly(false);
    ui->lineEdit_content->setPlaceholderText("ReadAndWrite");
}

void Widget::on_btn_ro_clicked()
{
    ui->lineEdit_content->setReadOnly(true);
    ui->lineEdit_content->setPlaceholderText("ReadOnly");
}

void Widget::on_btn_disable_clicked()
{
    ui->lineEdit_content->setEnabled(false);
    ui->lineEdit_content->setPlaceholderText("Disabled");
}

void Widget::on_btn_enable_clicked()
{
    ui->lineEdit_content->setEnabled(true);
    ui->lineEdit_content->setPlaceholderText("Enabled");
}

void Widget::on_btn_none_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setInputMask("");
}

void Widget::on_btn_phone_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setInputMask("99999999999;#"); //要求输入0-9的数字 ,未输入时显示#
}

void Widget::on_btn_key_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setInputMask(">AAAAA-AAAAA-AAAAA-AAAAA-AAAAA;#"); //要求输入大写字母 ,未输入时显示#
}

void Widget::on_btn_data_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setInputMask("0000-00-00"); //要求输入0-9的数字
}

void Widget::on_btn_ip_clicked()
{
    ui->lineEdit_content->clear();
    ui->lineEdit_content->setInputMask("000.000.000.000;_"); //要求输入0-9的数字,未输入时显示_
}

void Widget::on_lineEdit_num2_editingFinished()
{
    int sum = ui->lineEdit_num1->text().toInt() + ui->lineEdit_num2->text().toInt();
    ui->lineEdit_sum->setText(QString::number(sum));
}

void Widget::on_lineEdit_num4_textChanged(const QString &arg1)
{
    int minus = ui->lineEdit_num3->text().toInt()- arg1.toInt();
    ui->lineEdit_minus->setText(QString::number(minus));
}

void Widget::on_lineEdit_num6_returnPressed()
{
    int multi = ui->lineEdit_num5->text().toInt() * ui->lineEdit_num6->text().toInt();
    ui->lineEdit_muliti->setText(QString::number(multi));
}
