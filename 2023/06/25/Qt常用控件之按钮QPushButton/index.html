Qt常用控件之按钮QPushButton

本文最后更新于：2023年6月29日 晚上

QPushButton 是 Qt 中的按钮类，属于最常用的控件，当点击按钮时，就可执行相应的操作。

可以在UI设计界面直接拖动push button来创建按钮，也可以通过代码进行new创建，如 :

QPushButton *button = new QPushButton("&Download", this);

以上创建了一个带有文字 Download 的按钮，快捷键为 Alt + D, 这里的 & 符号为快捷键标识。

QpushButton的属性和方法

  • 获取和设置按钮的文本
QString text() const
void setText(const QString &text)
  • 获取和设置按钮的图标
QIcon icon() const
void setIcon(const QIcon &icon)
  • 样式表可以设置包括文本颜色、背景色、边框、字体等很多样式
// 只要继承自QWidget类的控件，都有该属性
QString styleSheet() const
void setStyleSheet(const QString &styleSheet)
  • Signals & Slots
void clicked(bool checked = false) //通常不需要传递这个默认参数
void pressed()
void released()
void toggled(bool checked)
    
void showMenu() //弹出菜单

其它函数具体参看帮助文档。

实例

下面进行功能实现：

void Widget::on_btn_stop_clicked()
{
    /* 获取按钮文本 */
    QString text = ui->btn_stop->text();
    /* 判断处理 */
    if(text == "启动")
    {
        ui->btn_stop->setText("停止");// 设置文本和字体
        ui->btn_stop->setIcon(QIcon(":/stop.ico"));// 设置图标和图标大小
        ui->btn_stop->setIconSize(QSize(32, 32));
    }
    else
    {
        ui->btn_stop->setText("启动");
        ui->btn_stop->setIcon(QIcon(":/start.ico"));
        ui->btn_stop->setIconSize(QSize(32, 32));
    }
}


void Widget::on_btn_connect_clicked()
{
    QString s = ui->btn_connect->text();
    if(s == "连接")
    {
        ui->btn_connect->setText("断开连接");
        ui->btn_send->setEnabled(true);
    }
    else
    {
        ui->btn_connect->setText("连接");
        ui->btn_send->setEnabled(false);
    }
}

/* 按钮信号演示 */
void Widget::on_btn_signal_clicked()
{
    qDebug() << "clicked...";
}


void Widget::on_btn_signal_pressed()
{
     qDebug() << "pressed...";
}


void Widget::on_btn_signal_released()
{
     qDebug() << "released...";
}

对于样式表：

在右侧的属性窗口中，或者在按钮上右键->【改变样式表…】，打开编辑样式表的对话框，设置了按钮的字体颜色和背景渐变‘RainBow’。
