Qt常用控件之滑动条QSlider

本文最后更新于：2023年7月3日 下午

QSlider 是滑动条控件，滑动条可以在一个范围内拖动，并将其位置转换为整数。

属性和方法

QSlider 绝大多数属性和方法都是从 QAbstractSlider 继承而来。

  • 获取和设置当前值
int value() const;
void setValue(int);
  • 获取和设置最大、最小值
int maximum() const;
void setMaximum(int);
int minimum() const;
void setMinimum(int);
void setRange(int min, int max)
  • 获取和设置滑动条的方向
Qt::Orientation orientation() const;
void setOrientation(Qt::Orientation);
  • 获取和设置步长
/* 键盘的左右箭头 */
int singleStep() const;
void setSingleStep(int);
/* 键盘的PageUp PageDown */
int pageStep() const;
void setPageStep(int);
  • 获取和设置刻度位置
QSlider::TickPosition tickPosition() const
void setTickPosition(QSlider::TickPosition position)
    
QSlider::NoTicks            //不显示刻度
QSlider::TicksBothSides     //标尺两侧都显示刻度
QSlider::TicksAbove         //标尺上方显示刻度
QSlider::TicksBelow         //标尺下方显示刻度
QSlider::TicksLeft          //标尺左侧显示刻度
QSlider::TicksRight         //标尺右侧显示刻度
  • 信号
void valueChanged()
void sliderPressed()
void sliderMoved()
void sliderReleased()

实例

新建工程，进入UI设计界面，设计如下：

功能实现：

/* 横向滑动条属性在UI界面已经设置，下面进行竖向滑动条属性设置 */
ui->slider_v->setRange(0, 100);
ui->slider_v->setSingleStep(1);
ui->slider_v->setPageStep(10);
ui->slider_v->setTickPosition(QSlider::TicksRight);
ui->slider_v->setTickInterval(5);

/* 数值改变将值显示在文本框中 */
connect(ui->slider_h, &QSlider::valueChanged, this, [=](){
    int value = ui->slider_h->value();
    ui->lineEdit_value->setText(QString::number(value));
});
connect(ui->slider_v, &QSlider::valueChanged, this, [=](){
    int value = ui->slider_v->value();
    ui->lineEdit_value->setText(QString::number(value));
});
