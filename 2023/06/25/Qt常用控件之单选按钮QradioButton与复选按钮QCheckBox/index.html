Qt常用控件之单选按钮QradioButton与复选按钮QCheckBox

本文最后更新于：2023年7月3日 下午

QRadioButton是单选按钮，常用作表单互斥选择选项，在“多选一”的场景中使用，也就是说在一组单选按钮中，一次只能选中一个单选按钮。如下所示：

QCheckBox是复选按钮，常用在”多选多“的场景中。如下所示：

属性和方法

QRadioButton和QCheckBox的属性和方法主要继承于父类 QAbstractButton 。

  • 获取和设置显示的文本
QString text() const
void setText(const QString &text)
  • 获取和设置单选按钮的选中状态
bool isChecked() const
void setChecked(bool)
  • 获取和设置自动互斥，单选按钮默认是互斥的，复选按钮默认不是互斥的。
bool autoExclusive() const
void setAutoExclusive(bool)

单选按钮“多选一” 要满足以下两个条件：

  1. 同一组的单选按钮，在同一个布局中。
  2. 单选按钮的 autoExclusive 属性设置为 true，默认是使能的。

  • 设置和获取复选按钮的状态
Qt::CheckState checkState() const
void setCheckState(Qt::CheckState state)

信号

QRadioButton的信号与槽同样主要继承父类 QAbstractButton 。

void clicked(bool checked = false)
void pressed()
void released()
void toggled(bool checked) //选中与未选中的状态变化

功能实现：

Widget::Widget(QWidget *parent)
    : QWidget(parent)
    , ui(new Ui::Widget)
{
    ui->setupUi(this);

    /* 设置国籍默认选项 */
    ui->radioButton_China->setChecked(true);

    /* 连接信号与槽 */
    connect(ui->checkBox_clibling, &QCheckBox::clicked, this, &Widget::checkBox_hobby_slot);
    connect(ui->checkBox_playball, &QCheckBox::clicked, this, &Widget::checkBox_hobby_slot);
    connect(ui->checkBox_reading, &QCheckBox::clicked, this, &Widget::checkBox_hobby_slot);
    connect(ui->checkBox_swimming, &QCheckBox::clicked, this, &Widget::checkBox_hobby_slot);

    connect(ui->btn, &QPushButton::clicked,[=](){
        btn_info_slot();
    });
}

void Widget::on_checkBox_all_clicked()
{
    if(ui->checkBox_all->isChecked())
    {
        ui->checkBox_clibling->setChecked(true);
        ui->checkBox_playball->setChecked(true);
        ui->checkBox_reading->setChecked(true);
        ui->checkBox_swimming->setChecked(true);
    }
    else
    {
        ui->checkBox_clibling->setChecked(false);
        ui->checkBox_playball->setChecked(false);
        ui->checkBox_reading->setChecked(false);
        ui->checkBox_swimming->setChecked(false);
    }
}

void Widget::checkBox_hobby_slot()
{
    /* 获取爱好是否选中的状态 */
    bool ischeck_clibling = ui->checkBox_clibling->isChecked();
    bool ischeck_playball = ui->checkBox_playball->isChecked();
    bool ischeck_reading = ui->checkBox_reading->isChecked();
    bool ischeck_swimming = ui->checkBox_swimming->isChecked();

    /* 全部选中 */
    if(ischeck_clibling && ischeck_playball && ischeck_reading && ischeck_swimming)
    {
        ui->checkBox_all->setCheckState(Qt::Checked);
    }
    /* 全部未选中 */
    else if(!ischeck_clibling && !ischeck_playball && !ischeck_reading && !ischeck_swimming)
    {
        ui->checkBox_all->setCheckState(Qt::Unchecked);
    }
    /* 部分选中 */
    else
    {
        ui->checkBox_all->setCheckState(Qt::PartiallyChecked);
    }
}

void Widget::btn_info_slot()
{
    QString str;
    /* 性别 */
    str = "性别：";
    if(ui->radioButton_male->isChecked())
        str += ui->radioButton_male->text();
    else
        str += ui->radioButton_female->text();
    /* 婚姻 */
    str += "; 婚姻状况：";
    if(ui->radioButton_lady->isChecked())
        str += ui->radioButton_lady->text();
    else
        str += ui->radioButton_woman->text();
    /* 学历 */
    str += "; 学历：";
    if(ui->radioButton_junior->isChecked())
        str += ui->radioButton_junior->text();
    else if(ui->radioButton_bachelor->isChecked())
        str += ui->radioButton_bachelor->text();
    else if(ui->radioButton_doctor->isChecked())
        str += ui->radioButton_doctor->text();
    else if(ui->radioButton_master->isChecked())
        str += ui->radioButton_master->text();
    /* 国籍 */
    str += "; 国籍：";
    if(ui->radioButton_America->isChecked())
        str += ui->radioButton_America->text();
    else if(ui->radioButton_China->isChecked())
        str += ui->radioButton_China->text();
    else if(ui->radioButton_France->isChecked())
        str += ui->radioButton_France->text();
    else if(ui->radioButton_German->isChecked())
        str += ui->radioButton_German->text();
    /* 爱好 */
    str += "; 爱好：";
    if(ui->checkBox_clibling->isChecked())
        str += ui->checkBox_clibling->text() + " ";
    if(ui->checkBox_playball->isChecked())
        str += ui->checkBox_playball->text() + " ";
    if(ui->checkBox_reading->isChecked())
        str += ui->checkBox_reading->text() + " ";
    if(ui->checkBox_swimming->isChecked())
        str += ui->checkBox_swimming->text() + " ";
    /* 信息输出 */
    QMessageBox::information(this, "您的个人信息：", str);
}
