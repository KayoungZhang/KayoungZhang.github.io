Qt Creator的使用

本文最后更新于：2023年6月14日 晚上

Qt Creator的设置

打开Qt Creator, 选择Tools -> Options 进入设置界面：

  • Environment：可以设置相应的主题和语言环境及UI颜色。

  • Text Editor：可以进行相应的文本编辑设置。

  • Build & Run：进行编译和运行的设置

新建Qt Widget项目

  • 依次点击 File -> New File or Project,选择Qt Widget Application

  • 填写工程名和选择存储路径，选择 qmake 编译，用 qmake 生成 Makefile 用于项目的编译。

  • 选择基类：

  • 选择目标平台的编译器：

  • 可选择版本控制：

  • 完成项目创建，可点击左下角绿色图标编译运行，此时会弹出一个空白框。

项目文件介绍

  • demo.pro: 项目管理文件。

还可以在其中设置应用程序其他信息：

#程序版本
VERSION = 1.0.0.0
#程序图标
RC_ICONS = client.ico
#公司名称
QMAKE_TARGET_COMPANY ="KayoungZhang"
#程序说明
QMAKE_TARGET_DESCRIPTION = "TCP Client"
#版权信息
QMAKE_TARGET_COPYRIGHT = "Copyright(C) 2022"
#程序名称
QMAKE_TARGET_PRODUCT = "app"
#程序语言
#0x0800代表和系统当前语言一致
RC_LANG = 0x0800

  • Header 分组，存放的是项目内所有的头文件*.h。

  • Source 分组，存放的是项目内的所有C++源码文件*.cpp。

  • Forms 分组，存放项目内所有界面文件*.ui。双击ui文件就可以进入图形设计界面，返回时则点击左边Edit图标。

在控件区，拖拽一个Label组件到中间的设计区，编辑文字内容为：“Hello，Qt！”，选择此 Lable 然后在其属性区，找到 geometry ,将 Width 改为 80， Height 改为 30，然后找到 font，将 Point Size设置为 12，勾选 Bold。

项目的编译和运行

单击左窗口的 Projects 图标，查看编译工具。

然后点击左下角编译运行：

至此完成一个简单的项目创建。

项目的发布

当项目全部完成后，就可以进行打包发行了。

选择Release后，编译运行完，打开项目工程目录，找到编译后的exe文件，双击运行，会发现运行不了，是因为缺少dll动态链接库。可以将Qt的bin目录添加到电脑系统的Path环境变量中，为了保证在其他电脑上也可以正常运行，需要找到相应的dll文件，Qt提供了一个 windeployqt 工具，可以自动创建可部署的文件夹，具体操作如下：

  1. 将生成的exe文件复制到一个空的文件夹中。

  1. 打开Qt的Min GW命令行，进入刚才的文件夹，然后输入命令：windeployqt后面跟程序名。

  1. 可以将部分不需要的内容删除，如下所示：

  1. 将精简后的程序压缩即可在其他同平台电脑运行了,也可以使用工具 “单文件制作封装工具(Enigma Virtual Box)”将其打包成独立版程序。如果需要整理成一个安装程序，就需要专用工具进行打包处理，一般用到的工具有：

    • HM NIS Edit（https://hmne.sourceforge.net/）
    • NSIS
    • Inno setup（https://jrsoftware.org/）
