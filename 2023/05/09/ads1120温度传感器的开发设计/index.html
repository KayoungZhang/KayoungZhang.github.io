ads1120温度传感器的开发设计

本文最后更新于：2023年5月11日 晚上

前面介绍了ds18b20单总线测量温度，一般应用在环境温度的测量，但开发过程中，针对高低温，宽范围的温度测量，此时用到最多的传感器一般是热电阻(RTDs)和热电偶，如PT100系列热电阻，K-type热电偶。在电路上通过放大器将测量的小信号放大输出，就可以得到测量的温度数据，当然比较简单的做法是使用集成芯片进行处理，像MAX6675，ADS1120，ADS1220等等，之前项目我选用的是ads1120芯片，下面详细介绍一下它的具体情况。

芯片特性

  • 电压： 2.3 V ~ 5.5 V
  • SPI接口 (Mode 1： CPOL=0，CPHA=1）
  • PGA: 1 V/V ~ 128 V/V
  • 采样率: 最大 2kSPS
  • 输入：4个单端输入口或两个差分输入口
  • 16 位无噪声分辨率（20SPS 时）
  • 内部基准电压： 2.048-V，漂移：5 ppm/°C
  • 功耗：1.4mW
  • 采用单周期稳定数字滤波器，在 20SPS 时实现 50Hz 和 60Hz 谐波抑制
  • 双匹配可编程电流源：50μA 至 1.5mA

功能框图

PGA共模电压要求

为了始终处于 PGA 的线性工作范围内，输入信号必须满足以下要求。可参考datasheet具体实例。

电压基准

该器件集成了 2.048V 低漂移电压基准。提供两对差分基准输入（REFP0/REFN0 和 REFP1/REFN1）。 基准源通过配置寄存器的两位 (VREF[1:0]) 进行选择。默认选择内部基准。上电后、退出掉电模式或由外部基准源切换至内部基准后，内部基准电压在 25µs 内即可完全稳定。

差分基准输入支持自由选择共模基准电压。REFP0 和 REFN0 为专用基准输入，而 REFP1 和 REFN1 分别与输入 AIN0 和 AIN3 共享。所有基准输入均在内部进行缓冲，以提高输入阻抗。因此，使用外部基准时，通常无需额外使用基准缓冲器。

在多数情况下，转换结果与基准源稳定性成正比。转换结果将反映电压基准的所有噪声和漂移。

激励电流源

该器件为 RTD 应用提供两个匹配的可编程激励电流源 (IDAC)。使用配置寄存器的相应位 (IDAC[2:0]) 可通过编程方式将电流源输出电流设定为 10μA、50μA、100μA、250μA、500μA、1000μA 或 1500μA。每个电流源均可与所有模拟输入 (AINx) 及所有专用基准输入（REFP0 和 REFN0）相连。两电流源还可与同一引脚相连。IDAC 的路由通过配置寄存器的位（I1MUX[2:0]，I2MUX[2:0]）进行配置。

电路设计

参考芯片datasheet的应用电路，我采用如下兼容设计，可自由切换外接热电偶和3线制PT100。

PCB设计

主要是模拟和数字地的隔离，电源引脚必须通过低 ESR 陶瓷电容旁路至接地。旁路电容的最佳位置应尽可能靠近电源引脚。采用差分连接的模拟输入之间必须放置差分电容。适用于差分测量的最佳输入组合包括：AIN0、AIN1 和 AIN2、AIN3，必须选择高品质差分电容。C0G (NPO) 电容属性稳定且具备低噪声特性，这是最适用的陶瓷贴片电容。对热电偶输入连接周围的覆铜区域进行热隔离，以构建热性能稳定的冷端。

温度传感器

ADS1120 集成了一个精密温度传感器。通过将配置寄存器的 TS 位置 1 可使能温度传感器模式。。温度数据以 14 位结果呈现，与 24 位转换结果左对齐。负数以二进制补码形式表示。数字代码转换温度公式：如果 MSB 为 0，将十进制代码乘以 0.03125°C 即可获得结果。如果 MSB = 1，则将结果减 1 后对所有位取补码。之后将结果乘以 –0.03125°C。

示例：器件回读 0960h: 0960h 的 MSB = 0。

0960h · 0.03125°C = 2400 · 0.03125°C = 75°C

示例：器件回读 3CE0h: 3CE0h 的 MSB = 1。

减 1 后对结果取补码：3CE0h → 0320h （C320h & 3FFFh）

0320h · (–0.03125°C) = 800 · (–0.03125°C) = –25°C

/**
  * @brief  MCU SPI initial
  * @param  None
  * @retval None
  */
void spi_init(void)
{
	ADS1120_RDY = 1;	
	ADS1120_CS  = 1;
	ADS1120_SDI = 0;	
	ADS1120_SDO = 0;
	ADS1120_CLK = 0;
}

/**
  * @brief  ads1120 initial config
  * @param  sensor_type: 传感器类型
  * @retval None
  */
void ads1120_confiuration(sensor_e sensor_type)
{
	/* ads1120 register configuration */
	uint8_t config[4] ={0, 0, 0, 0};

	switch (sensor_type)
	{	
		case PT100:
		{/* PT100 configuration */
			config[0] = ADS1120_MUX_0_1 | ADS1120_GAIN_8; /* register 0 configuration */
			config[1] = ADS1120_DR_20 | ADS1120_CC; /* register 1 configuration */
			config[2] = ADS1120_VREF_0 | ADS1120_REJECT_BOTH | ADS1120_IDAC_500; /* register 2 configuration */
			config[3] = ADS1120_IDAC1_AIN2 | ADS1120_IDAC2_AIN3; /* register 3 configuration */
			break;				
		}
		case K_TYPE:
		{/* K-type Thermocouple configuration */
			config[0] = ADS1120_MUX_0_1 | ADS1120_GAIN_32; /* register 0 configuration */
			config[1] = ADS1120_DR_20 | ADS1120_CC; /* register 1 configuration */
			config[2] = ADS1120_VREF_INT | ADS1120_REJECT_BOTH | ADS1120_IDAC_OFF; /* register 2 configuration */
			config[3] = ADS1120_IDAC1_OFF | ADS1120_IDAC2_OFF; /* register 3 configuration */	
			break;				
		}
		case INTER:
		{/* ads1120 internal temperature sensor configuration */
			config[0] = 0; /* register 0 configuration */
			config[1] = ADS1120_CC | ADS1120_TEMP_SENSOR; /* register 1 configuration */
			config[2] = 0; /* register 2 configuration */
			config[3] = 0; /* register 3 configuration */	
			break;				
		}
		default: break;
	}
	ADS1120_CS = 0;	
	soft_spi_rw_mode1(ADS1120_RESET); // reset ads1120
	Delay100ms();
	ads1120_wr_reg(ADS1120_0_REGISTER, 0x04, config); //config ads1120 register
	Delay100ms();
	soft_spi_rw_mode1(ADS1120_START); //start conversation
	ADS1120_CS = 1;
}

/**
  * @brief  gpio 模拟spi，只支持SPI模式1,CPOL=0，CPHA=1, MSB first下降沿采样
  * @param  byte: 发送的一字节数据
  * @param	read_dat:读取的一字节数据
  * @retval None
  */
uint8_t soft_spi_rw_mode1(uint8_t byte) 
{
	uint8_t i,read_dat=0;
 
	for(i = 0; i < 8; i++)     // 循环8次
	{
		/* 发送数据 */
		ADS1120_CLK = 1;     //拉高时钟
		if(byte & 0x80)
			ADS1120_SDI = 1;  //若最到位为高，则输出高
		else      
			ADS1120_SDI = 0;   //若最到位为低，则输出低
		byte <<= 1;     // 低一位移位到最高位
		Delay1us();
		/* 接收数据 */
		ADS1120_CLK = 0;     //拉低时钟
		read_dat <<= 1;     //数据左移
		if(ADS1120_SDO)
			read_dat++;     //若从从机接收到高电平，数据自加一
		Delay1us();
	}
	return (read_dat);     //返回数据
}

/**
  * @brief  ads1120读寄存器
  * @param  addr：寄存器开始地址
  *	@param	num：读取的字节数
  *	@param	p_data：读取的数据指针
  * @retval None
  */
void ads1120_rd_reg(uint8_t addr, uint8_t num, uint8_t* p_data)
{
	uint8_t i;
	
 	/* send the command byte */
	soft_spi_rw_mode1(ADS1120_RREG | (((addr << 2) & 0x0c) |((num - 1) & 0x03)));
  /* get the register content */
	for (i = 0; i < num; i++)
	{
		*p_data++ = soft_spi_rw_mode1(0x00);
	}
	return;
}
/**
  * @brief  ads1120写寄存器
  * @param  addr：寄存器开始地址
  *	@param	num：写入的字节数
  *	@param	p_data：写入数据的指针
  * @retval None
  */
void ads1120_wr_reg(uint8_t addr, uint8_t num, uint8_t* p_data)
{
	uint8_t i;
	
 	/* send the command byte */
	soft_spi_rw_mode1(ADS1120_WREG | (((addr << 2) & 0x0c) |((num - 1) & 0x03)));
  /* send the register content */
	for (i = 0; i < num; i++)
	{
		soft_spi_rw_mode1(*p_data++);
	}

	return;
}

/**
  * @brief  ads1120读取转换数据
  * @param  None
  * @retval result：读取到的两字节数据
  */
long ads1120_rd_data(void)
{
	long result = 0;

  /* assert CS to start transfer */
	ADS1120_CS = 0;
	Delay1us();
  /* send the command byte */
  soft_spi_rw_mode1(ADS1120_RDATA);//read conversation data
//	while(ADS1120_RDY); //等待数据到来
	result = soft_spi_rw_mode1(0x00);
	result = (result << 8) | soft_spi_rw_mode1(0x00);
	/* sign extend data */
  if (result & 0x8000)
		result |= 0xffff0000;
//		result = (~result) & 0x7fff;
//	uart1_send_byte(result >> 8);
//	uart1_send_byte(result);
	Delay1us();
	ADS1120_CS = 1;
	
  return result;
}
/**
  * @brief  ads1120获取温度
  * @param  temperature：获取的温度值
  * @param  sensor_type: 传感器类型
  * @retval None
  */
void ads1120_get_temperature(float* temperature, sensor_e sensor_type)
{
	long value = 0;
	float voltage = 0.0f;
	
	value = ads1120_rd_data();
	switch (sensor_type)
	{	
		case PT100:
		{/* PT100 */	
			voltage = (float)(value * 1.65f / 8 / 32768);
			*temperature = (voltage * 2000 - 100)/0.385f;//IDAC_500uA, PT100每升高1℃，阻值增加0.385 ohm
			break;				
		}
		case K_TYPE:
		{/* K-type thermocouple */	
//			uart1_send_byte(calibration_value >> 8);
//			uart1_send_byte(calibration_value);
			voltage = (float)(value * 62.5 / 32  + cold_compensation); //voltage=value*2.048f/32/32768*1000000;转换为uV
			*temperature = (float)(voltage / 41.0);	//K-type赛贝克系数：41℃/uV
			break;				
		}			
		case INTER:
		{/* ads1120 internal temperature sensor 14bit left-data */	
			value = (value >> 2) & 0x3fff;
			if(value & 0x2000) //negative temp
			{
				value = (~(value - 1))&0x1fff;	
				*temperature =(float)(value * (-0.03125f));
			}
			else
			{
				*temperature =(float)(value * 0.03125f);//postive temp	
			}		
			break;				
		}
		default: break;
	}
	return;
}

/**
  * @brief  ads1120平均温度
  * @param  temperature：温度平均值
  * @param  sensor_type: 传感器类型
  * @retval None
  */
void ads1120_get_average_temp(float* temperature, sensor_e sensor_type)
{
	uint8_t i;
	float sum = 0.0; //温度和
	
	for(i = 0; i < 8; i++)
	{
		ads1120_get_temperature(temperature, sensor_type);
		sum += *temperature;			
	}
	*temperature = (float)(sum / 8.0);
	sum = 0.0;
}

/**
  * @brief  ads1120 K型热电偶冷端补偿
  * @param  voltage：获取冷端电压值的指针（单位：uV）
  * @retval None
  */
void ads1120_thermocouple_compensation(int* voltage)
{
	/* ads1120 register configuration */
	uint8_t config[4] ={0, 0, 0, 0}; 
	long result = 0;
	float temp = 0;
	/* ads1120 internal temperature sensor configuration */
	config[0] = 0; /* register 0 configuration */
	config[1] = ADS1120_TEMP_SENSOR; /* register 1 configuration */
	config[2] = 0; /* register 2 configuration */
	config[3] = 0; /* register 3 configuration */		
	ADS1120_CS = 0;	
	soft_spi_rw_mode1(ADS1120_RESET); // reset ads1120
	Delay100ms();
	ads1120_wr_reg(ADS1120_0_REGISTER, 0x04, config); //config ads1120 register
	Delay100ms();
	soft_spi_rw_mode1(ADS1120_START); //start conversation
//  soft_spi_rw_mode1(ADS1120_RDATA);//read conversation data
	while(ADS1120_RDY); //等待数据到来
	result = soft_spi_rw_mode1(0x00);
	result = (result << 8) | soft_spi_rw_mode1(0x00);
	ADS1120_CS = 1;
	result = (result >> 2) & 0x3fff; //14bit postive temp	
	if(result & 0x2000) //negative temp
	{
		result = (~(result - 1))&0x1fff;	
		temp =(float)(result * (-0.03125f));
	}
	else //postive temp	
	{
		temp =(float)(result * 0.03125f);
	}	
	/* 根据表格数据得出电压与温度关系式：https://srdata.nist.gov/its90/main/its90_main_page.html */
	*voltage = (int)(40.309 * temp - 5.3031); //NIST- its90 温度与电压关系式（单位：uV）
	return;
}
