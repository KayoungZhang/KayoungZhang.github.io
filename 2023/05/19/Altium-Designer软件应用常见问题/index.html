Altium Designer软件应用常见问题

本文最后更新于：2023年5月24日 晚上

使用Altium Designer时，有时候会碰到一些不常见的问题，以下总结几个，后续发现了问题，再增加。

  • 高版本AD无法寻找系统设置项

解决方法：

高版本在软件右上角有个设置图标即是设置项或 Tools -> Preferences.

  • 在低版本AD，PCB中拖动元件，元件不显示。

解决方法：

  1. Preferences -> PCB Editor -> Display -> 去掉Use Alpha Blending对勾。
  2. Preferences -> PCB Editor -> Display -> 选中Use DirectX if possible。
  • 在PCB中元器件随时绿色报错，影响视觉。

解决方法：

  1. Tools -> Design Rule Check -> 除了Electrical，其他项对勾都去掉。
  2. Design->Rules -> Electrical Clearance -> 设置Minimum clearance 5mil.
  3. 快捷键T -> M，复位报错。

  • 原理图导入PCB中，总是有元件无法导入，可能是footprint的unique ID相同。

解决方法：复位零件ID。

  • 低版本AD中，原理图导出pdf文件，中文字体不显示/乱码

解决方法：

打开Preference界面，System->General，将系统字体改为中文；如果还不显示，可能原理图之前的文字没有修正过来，需要手动修改。

  • PCB编辑界面左上角悬浮框怎么取消

解决方法：

按快捷键Shift+H，可以选择打开或者关闭抬头显示；

按快捷键Shift+G，抬头跟随鼠标移动，再次按下Shift+G即可固定。

  • 原理图库引脚上面一横怎么添加的（表示低电平有效）

解决方法：

  • pcb设计中某个零件需要铺铜，铺完后发现不理想，不想重新铺铜，怎么操作。

解决方法：

  1. 低版本，使用快捷键 M + G进行修铜即可。
  2. 高版本，直接通过鼠标选择边框进行修改，然后重新铺铜。
