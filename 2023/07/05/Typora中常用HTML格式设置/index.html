1.颜色

颜色的种类有很多，HTML和CSS中的颜色规格有140多个（17种标准颜色，再加130多个其它颜色）。17种标准颜色：

浅绿色

orange 		紫色
purple 		红色

  • Hex颜色代码： <span style="color:#FF0000">Red text</span>

  • HTML颜色名称: <span style="color:red">Red text</span>

  • RGB颜色代码: <span style="color:rgb(255,0,0)">Red text</span>

  • HSL颜色代码： <span style="color:hsl(0,100%,50%)">Red text</span>

    色相值：0-360，饱和度：0~100%，亮度：0~100% 。

以上四种方式都能实现以下字体样式：

Red text

  • RGBA颜色代码：<span style="color:rgba(255,0,0,0.5)">Red text</span>

    a：0~1。0表示完全透明，1表示完全不透明

Red text

2.字体

20号微软雅黑加粗字体，绿色背景

3.链接设置

  • 普通：<a href="www.baidu.com">百度</a> ---> 百度

  • 颜色：<a href="www.baidu.com" style="color:OrangeRed">百度</a> ---> 百度

  • 清除下划线：<a href="www.baidu.com" style="text-decoration:none">百度</a> ---> 百度

4.其它

  • 换行：<br>

  • 键盘按键组合：如：<kbd>Ctrl</kbd>+<kbd>F9</kbd> ---> Ctrl+F9

  • 带拼音文字：如：<ruby>中国<rt>zhongguo</rt></ruby> ---> 中国zhongguo

  • 内嵌图片和音视频:

    • <img src="xxx.png" />
    • <audio src="xxx.mp3" />
    • <video src="xxx.mp4" />

  • 对齐:

    • <p align="center";>段落中心对齐</p>

    段落中心对齐

    • <center>内容中心对齐</center>

      内容中心对齐
