从零开始写C语言链表

本文最后更新于：2023年7月27日 晚上

链表的概念

链表（Linked List）是一种物理存储结构上非连续、非顺序的存储结构，数据元素（节点）的逻辑顺序是通过链表中的指针链接次序实现的 。

相比数组这种在定义即分配一段连续且固定空间大小的数据结构，链表可以根据需要动态分配离散的内存空间，解决了数组插入或删除元素需要移动大量元素的弊端，不过它的缺点就是需要牺牲更大的内存空间。一般链表在操作系统里面应用比较广泛。

链表的结构

链表由一系列 节点（node） 组成，每个节点包括两个部分：

  • 数据元素的 数据域
  • 下一个节点的 指针域

先来认识一个完整的简单链表，如下图所示：

  • 头指针：一个普通的指针，用于指明链表的位置，永远指向链表第一个节点的位置；
  • 头节点：通常头节点不是必须的，它的存在主要是为了方便对首元节点前做插入、删除操作的目的；
  • 首元节点：链表中第一个存有数据的节点为首元节点，只是一个称谓而已，没有实际意义；

头节点和头指针的主要区别：头指针只声明而没有分配存储空间，头节点需要声明并分配一个节点的实际物理内存。

链表的分类

  • 单向链表
  • 双向链表
  • 环形链表

单向链表的操作

以上直观了解了链表的实现方式，那么怎么来创建和操作一个链表呢？首先看单向链表，我们需要定义它的数据结构，具体实现代码为：

typedef struct Node {
    int data;       //数据类型，可以把int型的data换成任意数据类型，包括结构体struct等复合类型
    struct Node *next;  //单链表的指针域
}node_t;

单向链表的创建

单向链表的数据结构定义好后，就可以着手创建链表了，创建单链表的节点，主要分以下步骤:

  • 给当前的每个节点的数据结构分配空间大小
  • 将每个节点数据清零（分配的堆内存空间不一定是干净的）
  • 给节点初始化数据
  • 将该节点的指针域设置为NULL
/* 创建一个节点的函数 */
node_t *linkedlist_create(int data)
{
	node_t *node = (node_t *)malloc(sizeof(node_t));//1、定义一个头指针，然后分配内存空间
	if(node == NULL){
		printf("malloc error!\n");
    return NULL;
	}
	memset(node,0,sizeof(node_t));//2、清一下内存
	node->data = data;  //3、给链表节点的数据赋值
	node->next = NULL;  //4、将链表的指针域指向空
	return node;
}

遍历单向链表（遍历，修改，查找）

进行遍历，修改，查找的思路极其简单，只需要建立一个指向链表L的节点，然后沿着链表L逐个向后搜索即可。

//打印链表数据
void linkedlist_print(node_t* pH)
{
  node_t *temp = pH->next; //临时指针指向第一个节点 
  int i = 0;
  while(NULL != temp)//如果当前位置节点不为空
  {
    printf("第%d个元素的值为:%d\n", ++i, temp->data);
    temp = temp->next;
  }  
}
//链表内容的修改，链表中值为val1的元素变为val2。
node_t* linkedlist_modify(node_t* pH,int val1,int val2) 
{
  node_t *p = pH->next;
  int i = 0;
  while(p)
  {
    if(p->data == val1)
    {
      p->data = val2;
    }
    p = p->next;
  }
  return pH;
}
//p为原链表，elem表示被查找元素
int linkedlist_find(node_t* p, int elem) {
    int i = 1;
    
    p = p->next;//带头节点，p 指向首元节点
    while (p) {
        if (p->data == elem) {
            return i;
        }
        p = p->next;
        i++;
    }
    return -1;//返回-1，表示未找到
}

单向链表插入操作

链表的插入操作主要有三种情况：

  • 头部插入
  • 中部插入
  • 尾部插入

不管是哪种方式，链表插入节点操作主要分为查找到第i个位置，将该位置的next指针修改为指向新插入的节点，而新插入的节点next指针指向(i+1)个位置的节点。其操作方式可以设置一个临时节点，利用循环找到第i个位置，再进行插入。

/* 链表的头部插入 */ 
void linkedlist_top_insert(node_t *pH , node_t *new)
{
	node_t *p = pH ;//获取当前的位置head 
	new->next = p->next ;//新的节点的下一个节点设置为原来头节点的下一个节点
	p->next = new ;//原来的头节点的下一个节点设置为现在新插入的头节点 
}

/* 链表的中部插入 */
void linkedlist_tail_insert(node_t *pH , node_t *new, int pos)
{
	node_t *p = pH ; //获取当前的位置 
	int i = 0;
	for(i = 1; i < pos; i++)
	{
		p = p->next ;//查找第pos个位置的节点 
	}
	p->next = new ;//此时把新插入的节点赋值给这个位置 
}

/* 链表的尾部插入 */
void linkedlist_tail_insert(node_t *pH , node_t *new)
{
	node_t *p = pH ; //获取当前的位置head 
	
	while(NULL != p->next)//如果当前位置的下一个节点不为空 
	{
		p = p->next ;//移动到下一个节点 
	}
	p->next = new ;//此时把新插入的节点赋值给这个位置 
}

单向链表删除操作

删除元素要建立一个前驱节点和一个当前节点，当找到了我们需要删除的数据时，直接使用前驱节点跳过要删除的节点指向要删除节点的后一个节点，再将原有的节点通过free函数释放掉。

//单链表的删除，在链表中删除值为val的元素
node_t* linkedlist_remove(node_t* L,int val) {
    node_t *p,*pre;                   //pre为前驱节点，p为查找的节点。
    p = L->next;
     
    while(p->data != val) {              //查找值为x的元素
        pre = p;
        p = p->next;
    }
    pre->next = p->next;          //删除操作，将其前驱next指向其后继。
    free(p);
     
    return L;
}

双向链表的操作

双向链表的结构体程序如下：

typedef struct line{
    int data;           //data
    struct line *pre;   //previous node
    struct line *next;  //next node
}line;

双向链表的创建

对于创建双向链表，我们需要先创建头节点再逐步的进行添加，请注意，双向链表的头节点是有数据元素的，这与一般的单链表是不同的。

开辟一段新的内存空间作为新的节点，为这个节点进行的data进行赋值，然后将已成链表的上一个节点的next指针指向自身，自身的pre指针指向上一个节点。

line* initLine(line * head){
    int number,pos=1,input_data;
    //三个变量分别代表节点数量，当前位置，输入的数据
    printf("请输入创建节点的大小\n");
    scanf("%d",&number);
    if(number<1){return NULL;} //输入非法直接结束
    //////头节点创建///////
    head=(line*)malloc(sizeof(line));
    head->pre=NULL;
    head->next=NULL;
    printf("输入第%d个数据\n",pos++);
    scanf("%d",&input_data);
    head->data=input_data;
  
    line* list=head;
    while (pos<=number) {
        line * body=(line*)malloc(sizeof(line));
        body->pre=NULL;
        body->next=NULL;
        printf("输入第%d个数据\n",pos++);
        scanf("%d",&input_data);
        body->data=input_data;
        
        list->next=body;
        body->pre=list;
        list=list->next;
    }
    return head;
}

双向链表的遍历

如同单链表的遍历一样，利用next指针逐步向后进行索引即可。

void printLine(line* head){
    line *list = head;
    int pos=1;
    while(list){
        printf("第%d个数据是:%d\n",pos++,list->data);
        list=list->next;
    }
}

双向链表的插入操作

对于双向链表的插入操作，也有三种形式：头部插入、中部插入和尾部插入。操作方式大同小异，首先创建一个独立的节点，将其的pre指针指向所需插入位置的前一个节点，同时，其所需插入的前一个节点的next指针修改指向为该新的节点，同理，该新的节点的next指针指向原本的下一个节点，而修改下一个节点的pre指针指向创建的新节点，如下图所示：

line* insertLine(line* head,int data,int add){
    //三个参数分别为：进行此操作的双链表，插入的数据，插入的位置
    //新建数据域为data的节点
    line* temp=(line*)malloc(sizeof(line));
    temp->data=data;
    temp->pre=NULL;
    temp->next=NULL;
    //插入到链表头，要特殊考虑
    if (add==1) {
        temp->next=head;
        head->pre=temp;
        head=temp;
    }else{
        line * body=head;
        //找到要插入位置的前一个节点
        for (int i=1; i<add-1; i++) {
            body=body->next;
        }
        //判断条件为真，说明插入位置为链表尾
        if (body->next==NULL) {
            body->next=temp;
            temp->pre=body;
        }else{
            body->next->pre=temp;
            temp->next=body->next;
            body->next=temp;
            temp->pre=body;
        }
    }
    return head;
}

双向链表的删除操作

删除操作的过程是：首先将需要删除节点的前一个节点的next指针指向自己的下一个节点，同时将下一个节点的pre指针修改指向为自己的上一个节点，然后删除节点，释放空间。

line* deleteLine(line* head, int data){
    //输入的参数分别为进行此操作的双链表，需要删除的数据
    line* list=head;
    //遍历链表
    while (list) {
        //判断是否与此元素相等
        //删除该点方法为将该节点前一节点的next指向该节点后一节点
        //同时将该节点的后一节点的pre指向该节点的前一节点
        if (list->data==data) {
            list->pre->next=list->next;
            list->next->pre=list->pre;
            free(list);
            printf("--删除成功--\n");
            return head;
        }
        list=list->next;
    }
    printf("Error:没有找到该元素，没有产生删除\n");
    return head;
}
