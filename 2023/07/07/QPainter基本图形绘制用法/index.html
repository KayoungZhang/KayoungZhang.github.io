QPainter基本图形绘制用法

本文最后更新于：2023年7月7日 下午

QPainter概念

QPainter 是用于执行绘图操作的类，它可以绘制简单的点、线，也可以绘制饼图、文本、图像等复杂的形状。

QPainter 一般在绘图事件 paintEvent() 中进行绘制，当窗口需要改变或者重新绘制时，当使用 repaint()和 update() 后，就会自动调用成员函数 paintEvent()。

QPainter 的用法，简单的几个步骤:

  • 1.构造一个 QPainter 对象

  • 2.设置画笔，画刷等

  • 3.绘制图形

  • 4.销毁绘图工具

Qpainter主要属性

  • font()用于绘制文本的字体。
  • brush()定义用于填充形状的颜色或图案。
  • pen()定义用于绘制的线条或边框，也可定义线条颜色。
  • viewport()、window()、worldTransform() 构成了painter的坐标转换系统。

Qpainter绘图

QPainter提供了绘制大多数基本图形的函数，如drawPoint()、drawLine()、drawRect()等，所有这些函数都有多个整数和浮点版本（推荐使用浮点数版本）。下面对常用的绘图功能进行说明：

  • 绘制点
void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event); //不使用event事件
    QPainter painter(this);
    painter.drawPoint(10, 10);
    painter.drawPoint(QPoint(20, 20));
    painter.drawPoint(QPointF(30, 30));

    QPoint points[3] = {QPoint(50,50),QPoint(60,60),QPoint(70,70)};
    painter.drawPoints(points, 3);
}
  • 绘制线
void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event); //不使用event事件
    
    QPainter painter(this);
    painter.setPen(QColor(Qt::red));//设置画笔颜色
    
    QLine line(0, 0, 100, 100);
    painter.drawLine(line);
    painter.drawLine(0,100, 100, 0);
    painter.drawLine(QPoint(50, 0), QPoint(50, 100));

    QVector<QLine> vectorLine;
    vectorLine << QLine(10, 0, 10, 100);
    vectorLine << QLine(30, 0, 30, 100);
    vectorLine << QLine(50, 0, 50, 100);
    vectorLine << QLine(70, 0, 70, 100);
    vectorLine << QLine(90, 0, 90, 100);
    painter.drawLines(vectorLine);
}
  • 绘制矩形
void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event); //不使用event事件

    QPainter painter(this);
    painter.setPen(QColor(10,60,200));//设置画笔颜色
    painter.setBrush(QColor(200, 100, 100));//设置画刷颜色
    painter.drawRect(50, 50, 150, 150);

    QRect rect(QPoint(160, 160),QPoint(220,220));
    painter.drawRect(rect);
}

  • 绘制圆弧

0°在时钟三点的位置，圆弧是逆时针转动，起始角度和跨度角度都必须是度数的1/16，如：整个圆的角度就是 **16*360**.

void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event); //不使用event事件

    QPainter painter(this);

    QRectF rectangle(50.0, 50.0, 60.0, 60.0);

    int startAngle = 30 * 16;
    int spanAngle = 130 * 16;

    painter.drawArc(rectangle, startAngle, spanAngle);
    painter.setPen(QPen(Qt::blue,3,Qt::DotLine));
    painter.drawRect(rectangle);
}

  • 绘制椭圆
void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event);

    QPainter painter(this);
    painter.setRenderHint(QPainter::Antialiasing, true);//抗锯齿
    painter.drawEllipse(QPointF(120, 60), 50, 20);

    painter.setBrush(QColor(255, 160, 90)); //设置画刷颜色
    painter.drawEllipse(QPointF(120, 140), 40, 40); //绘制圆
}

  • 绘制多边形
void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event);

    QPainter painter(this);
    painter.setRenderHint(QPainter::Antialiasing, true);//抗锯齿

    QPointF points[4] = {QPointF(30, 40), QPointF(60, 150), QPointF(150, 160), QPointF(220, 100)};

    painter.drawPolygon(points, 4);
}

  • 绘制文本
void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event);

    QPainter painter(this);
    painter.setPen(QColor(0, 160, 230));//设置画笔颜色
    QFont font;//设置字体属性
    font.setFamily("Microsoft YaHei");//字体样式
    font.setPointSize(50);//字体点大小
    font.setItalic(true);//字体斜体
    painter.setFont(font);
    painter.drawText(rect(), Qt::AlignCenter, "QPainter");//绘制文本
}

  • 绘制图像

绘制图像函数有drawPixmap()、drawImage()，drawPixmap在屏幕设备上更快，drawImage()在QPrinter或其他设备上可能更快。

void Widget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event);

    QPainter painter(this);

    QPixmap pixmap;
    pixmap.load(":/pic/a.png");
    painter.drawPixmap(0, 0, 150, 150, pixmap);
    QImage image;
    image.load(":/pic/b.png");
    painter.drawImage(160, 0, image);
}
