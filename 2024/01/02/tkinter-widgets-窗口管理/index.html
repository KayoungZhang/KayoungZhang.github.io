tkinter widgets 窗口管理

本文最后更新于：2024年1月2日 晚上

Tkinter 多个控件被摆放到父窗口时，就需要进行布局管理（geometry management），tkinter中提供了三种管理方法，分别如下：

  • pack()

按照控件的添加顺序从上到下依次排列，通过代码可以直观知道摆放情况。

  • grid()

以行列网格形式进行排列。

  • place()

精确指定控件的摆放位置。

pack()方法

属性参数有：

属性 说明
anchor 控件在窗口中的对齐方位："n"/"w"/"s"/"e"/"ne"/"nw"/"sw"/"se"/"center"
expand 控件是否随着窗口扩展，参数值： True / False，默认为 False
fill 控件可在水平/垂直/同时在两个方向上进行拉伸，参数值： X/Y/BOTH/NONE
ipadx,ipady 控件边框与内容的距离（内边距），单位为像素
padx,pady 控件之间的上下、左右的距离（外边距），单位为像素
side 控件在窗口的位置，参数值： 'top','bottom','left','right'。

grid()方法

属性参数有：

属性 说明
column 控件位于表格中列数，默认为第 0 列
columnspan 控件所跨的列数，默认为 1 列，通过该参数可以合并一行中多个单元格。
ipadx,ipady 用于控制内边距，在单元格内部，左右、上下方向上填充指定大小的空间。
padx,pady 用于控制外边距，在单元格外部，左右、上下方向上填充指定大小的空间。
row 控件位于表格中行数，默认为第 0 行
rowspan 控件所跨的行数，默认为 1 行，通过该参数可以合并一列中多个单元格。
sticky 功能类似于 anchor，参数：N/S/W/E,即上下左右对齐，或组合使用，默认居中对齐。

place()方法

属性参数有：

属性 说明
anchor 控件在窗口中的对齐方位："n"/"w"/"s"/"e"/"ne"/"nw"/"sw"/"se"/"center"
x,y 定义控件在窗体中左上方位置
relx,rely 相对于父窗口的位置比例，取值范围： 0.0~1.0
height,width 控件的高度和宽度（单位为像素）
relheight,relwidth 相对于父窗口的大小比例，取值范围： 0.0~1.0

注意：在一个父窗口中，place()可以和grid()或pack()方法混合使用，但grid()和pack()方法不能混合使用。

一般在应用中使用pack()和grid()方法比较多。
