tkinter 事件

本文最后更新于：2024年1月15日 上午

GUI程序是一种事件导向的应用程序设计，事件的来源可能是有鼠标操作、键盘操作。tkinter提供有一些处理方式。

事件的绑定方式

  • 单个控件绑定方法
w.bind(sequence=None, func=None, add=None)
  • 所有控件绑定方法
w.bind_all(sequence=None, func=None, add=None)
  • 某类控件绑定方法
w.bind_class(className, sequence=None, func=None, add=None)

事件组合顺序（Event sequence）

tkinter中，事件表示方式如下：

<[事件描述符-]事件类型[-具体信息]>

其中：事件描述符（Event modifier）有如下：

描述符名称 描述
Alt Alt按键
Any 任意按键
Control Ctrl按键
Double 双击按键
Lock CapsLock按键
Shift Shift按键
Triple 三击按键

事件类型（Event type）有如下：

事件类型名 描述
Activate 激活控件
Button 按键
ButtonRelease 释放按键
Configure 改变控件大小和位置
Deactivate 禁止控件
Destroy 销毁控件
Enter 将鼠标移入光标可显示控件
Expose 控件被遮盖后显示
FocusIn 输入焦点移入控件
FocusOut 输入焦点移出控件
KeyPress 按键按下
KeyRelease 按键释放
Leave 鼠标光标移出控件
Map 控件映射，可在程序中显示
Motion 鼠标光标在控件内移动
MouseWheel 鼠标滚轮
Unmap 控件取消映射，不再在程序中显示
Visibility 程序窗口在屏幕上显示

具体信息有：

  • 鼠标按键
    • 1 （左键）
    • 2 （中键）
    • 3 （右键）
  • 键盘按键
    • 单个按键 A ~ Z
    • 按键名 ，如下：
.keysym Key .keysym Key .keysym Key
Alt_L 左Alt按键 KP_Multiply 小键盘按键 X Prior PageUp
Alt_R 右Alt按键 KP_Add 小键盘按键 + Next PageDown
Shift_L 左Shift按键 KP_Begin 小键盘中心按键 Pause Pause
Shift_R 右Shift按键 KP_Decimal 小键盘按键 . Print PrintScreen
Control_L 左Ctrl按键 KP_Delete 小键盘按键 Delete Cancel Break
Control_R 右Ctrl按键 KP_Divide 小键盘按键 / Scroll_Lock ScrollLock
Up ↑ KP_Subtract 小键盘按键 - Tab Tab
Left ← KP_End 小键盘按键 End Insert Insert
Down ↓ KP_Enter 小键盘按键 Enter End end
Right → KP_Home 小键盘按键 Home Escape Esc
F1 ~ F12 功能按键F1 ~ F12 KP_Insert 小键盘按键 Insert Num_Lock Num Lock
KP_0 ~ KP_9 小键盘按键 0 ~ 9 KP_Next 小键盘按键 PageDown Home Home
KP_Right *小键盘按键 → KP_Prior 小键盘按键 PageUp Return Enter
KP_Up 小键盘按键 ↑ KP_Down 小键盘按键 ↓ Delete Delete
KP_Left 小键盘按键 ← BackSpace Backspace Caps_Lock Caps Lock

举例如下：

<KeyPress-H> # 按下按键H
<Button-1> # 单击鼠标左键
<Control-Shift-KeyPress-H>  # control + shift + H  
<ButtonRelease-1> # 放开鼠标左键
<Double-Button-3> # 双击鼠标右键

绑定事件，如：

btn.bind('<Button-1>', event_handler) # 在按钮上单击鼠标左键，执行中断函数

    """获取光标位置"""
    pos_x = event.x
    pos_y = event.y
